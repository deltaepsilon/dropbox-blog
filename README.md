#Why?

- Because Markdown is than WYSIWYG html editors.
- Because there are all sorts of static blog generators out there, but none that let you work straight from [Editorially](https://editorially.com/).
- Because getting indexed is overrated.

#Install

Install your nodes. ```npm install```
Install your bowers. ```bower install```
Start the devs. ```grunt server```
Test it. ```Grunt test```
Build it. ```grunt build```
